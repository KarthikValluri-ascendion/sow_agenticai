# Title: Delivery Excellence - Project Circle Framework v2.1

## Source Reference
**Internal Governance:** DelEx AI Engine / Ascendion Delivery Standards

---

## Overview
> All new Ascendion projects must be classified into one of three **Process Circles**. This classification determines the specific automation toolkit and the depth of compliance required for successful delivery.

---

## Process Circles Classification

### 🟢 Circle: Alpha (Strategic Managed Services)
**Designed for high-touch, long-term strategic partnerships.**
* **Intake Requirement:** Comprehensive **Data Architecture & Ethics Gate**.
* **Ongoing Reporting:** Mandatory **Weekly Client Sentiment Analysis** or **Weekly Status Reports**.

### 🟡 Circle: Beta (Accelerated Delivery)
**Designed for rapid execution and scaling projects.**
* **Intake Requirement:** Implementation of **Dynamic Planning** modules.
* **Ongoing Monitoring:** Continuous **Automated Anomaly Detection** tracking sprint velocities and resource burn.

### 🔴 Circle: Gamma (Ad-hoc Tech Consulting)
**Designed for short-term, specialized technical advisory.**
* **Intake Requirement:** Minimalist governance package.
* **Core Guardrails:** Strict adherence to **AI Hygiene Guardrails** to guarantee data privacy and ethical tool usage.

## 🛑 Mandatory Action: Compliance Gate
Project Managers (PMs) **must** run the **Automated Compliance Scanning** tool prior to the formal project kickoff meeting.

*⚠️ **Note:** Failure to complete this scan will automatically block the project's active status in the DelEx portal.*

## Classification Rules (for automated SOW audit)

Use these **signals** to assign **Alpha**, **Beta**, or **Gamma** from the SOW text (exact labels optional; intent matters):

| If the SOW emphasizes… | Lean toward |
| :--- | :--- |
| Long-term strategic partnership, managed services, high-touch governance, enterprise data architecture, ethics review gates | **Alpha** |
| Rapid scaling, sprint velocity, resource burn, execution speed, integration of planning/automation modules | **Beta** |
| Short-term advisory, specialized consulting, limited scope, minimalist governance, hygiene/ethics as primary guardrail | **Gamma** |

**PASS (Agent 2):** After you choose the circle, the SOW must **substantially address** that row’s **Core Requirement** and **Key Monitoring & Guardrails** from the matrix (synonyms and clear paraphrases count; not every word must appear verbatim).

**PASS (Agent 3 / Compliance Gate):** The SOW should acknowledge **Automated Compliance Scanning** (or equivalent: automated compliance tooling, pre-kickoff compliance scan, DelEx compliance gate) aligned with the **Mandatory Action** above.
